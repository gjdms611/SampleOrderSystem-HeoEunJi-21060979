# Phase 29: 시료 검색 키워드 입력 안내 예시 추가

계층: view

## 배경

PRD 5.1(시료관리 조회/검색 동작)에 "검색 키워드 안내에는 시료 이름의 일부를 입력하면 된다는 설명과 예시를 함께 보여준다"가 명시됐는데, 현재 `prompt_search_keyword()`는 아무 설명 없이 "검색 키워드: "만 물어본다.

## 목표

검색 키워드 입력 프롬프트에 "시료 이름의 일부를 입력하세요" 같은 안내와 예시(`Wafer-A`, `Wafer`, `Chip` 등)를 함께 보여준다.

## 설계

- `view/console_view.py`의 `prompt_search_keyword()` 프롬프트 문구를 다음과 같이 바꾼다: 안내 문구 + 예시를 포함한 멀티라인 프롬프트로 변경. 로직/반환값(문자열 그대로 반환)은 바뀌지 않는다.

## 완료 조건

- [x] `python main.py`로 시료 검색 메뉴 진입 시 안내+예시가 함께 보이는지 수동 확인
